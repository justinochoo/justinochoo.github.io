Projects

Automatic Brain Segmentation Editor: The automatic segmentation algorithm (designed to define white and gray matter volumes from MRI data) deployed in FreeSurfer v6 is known to generate errors in which patches of white matter are left exposed. In the past, my lab has employed research assistants to fix these issues manually, a process which takes roughly an hour per brain. I have written a Python program to do the same task with greater accuracy in < 5 minutes. This program required BFS in 3D and optimization.

Research Experience

Vision and Perception Neuroscience Lab: In the VPNL, I'm currently working on a retinotopy project to define new fovea in human visual cortex. A novel stimuli set developed in my lab has generated particularly robust fMRI data in deep areas of the ventral and lateral visual streams. My workflow involves processing fMRI data (MATLAB), reorganizing data (C), and analyzing fMRI and anatomical data in FreeSurfer.
